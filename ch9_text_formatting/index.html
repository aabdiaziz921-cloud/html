HTML Text Formatting Elements

This page shows examples of common HTML text formatting tags.

Formatted Text HTML Element
This text is bold <b>Bold</b>
This text is important <strong>Important Text</strong>
This text is italic <i>Italic</i>
This text is emphasized <em>Emphasized</em>
This text is underlined <u>Underline</u>
This text is highlighted <mark>Highlighted</mark>
This is small text <small>Small Text</small>
This is deleted text <del>Deleted Text</del>
This is inserted text <ins>Inserted Text</ins>
H2O <sub>Subscript</sub>
x2 <sup>Superscript</sup>
